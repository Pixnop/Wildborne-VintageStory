# Dossier Mods

Placez tous les fichiers `.zip` des mods dans ce dossier.

## Contenu attendu

Tous les mods du modpack Wildborne (97 mods au total).

## Pour créer une release

1. Copiez tous les fichiers `.zip` des mods dans ce dossier
2. Allez sur GitHub Actions
3. Lancez le workflow "Create Modpack Release"
4. Entrez le numéro de version

Le workflow créera automatiquement une archive et une release GitHub.
