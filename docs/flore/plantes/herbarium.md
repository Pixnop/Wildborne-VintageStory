## Description

**Herbarium** est une bibliothèque réutilisable de code pour plantes, développée pour le mod Wildcraft. Elle fournit les mécaniques de base pour les autres mods botaniques du modpack.

### Systèmes fournis
- **Croissance des baies** : floraison, maturation, chute des feuilles
- **Bouturage et greffage** : propagation des plantes
- **Buissons hauts** : jusqu'à 4 blocs de hauteur
- **Vignes grimpantes** : avec supports de croissance
- **Restrictions de température** : croissance dépendante du climat

---

## Système de collection

### Herbier personnel
- **Carnet d'herbier** pour cataloguer vos découvertes
- Chaque plante peut être **pressée** et conservée
- Débloquez des **informations** sur les plantes collectées
- Suivez votre **progression** de découverte

### Étapes
1. **Trouvez** une nouvelle plante
2. **Récoltez** un spécimen
3. **Pressez**-le dans votre herbier
4. **Étudiez** pour débloquer ses propriétés
