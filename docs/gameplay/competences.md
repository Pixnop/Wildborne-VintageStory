## XSkills

Chaque action vous fait progresser. L'expérience accumulée se convertit en points de capacité pour débloquer des aptitudes spécialisées.

### Branches de compétences

| Compétence | Aptitudes clés |
|------------|----------------|
| Minage | Extraction de veines entières, excavation rapide |
| Bûcheronnage | Abattage efficace, récolte de résine |
| Agriculture | Rendements doublés, croissance accélérée |
| Combat | Maîtrise par arme (épées, masses, lances...) |
| Cuisine | Plats plus nutritifs, effets prolongés |
| Forge | Outils de qualité supérieure, durabilité |
| Élevage | Apprivoisement facilité, reproduction |

### Système de qualité
Les outils et équipements craftés peuvent avoir une **qualité variable** affectant leur durabilité et efficacité. Plus votre compétence est haute, meilleure sera la qualité.

## Trait Acquirer

![Trait Acquirer](https://moddbcdn.vintagestory.at/tasc6_d62e8ddc41f6f62988140b0fc7d77224.png){ align=right width=120 }

Vos actions façonnent votre personnage avec des traits permanents.

### Exemples de traits
| Action | Trait possible |
|--------|----------------|
| Manger varié | Estomac de fer |
| Combattre souvent | Réflexes aiguisés |
| Explorer longtemps | Endurance accrue |
| Miner beaucoup | Vision nocturne |

Attention : certains comportements peuvent aussi donner des traits **négatifs** !

## The Working Classes

![The Working Classes](https://moddbcdn.vintagestory.at/WorkingClassImage_a36ae338e09341d18dc7fa1c923d53ba.png){ align=right width=120 }

Choisissez un métier pour des bonus spécialisés dès le départ.

| Classe | Spécialité |
|--------|------------|
| Forgeron | +20% durabilité des outils forgés |
| Fermier | +15% rendement des récoltes |
| Chasseur | +25% dégâts aux animaux sauvages |
| Mineur | +10% chance de minerai bonus |
| Artisan | Crafts de meilleure qualité |

## Conseils

!!! tip "Spécialisation"
    Focalisez-vous sur quelques compétences au début pour progresser plus vite. Vous pourrez diversifier plus tard.

!!! tip "Multijoueur"
    En serveur, répartissez les spécialités entre joueurs pour une économie collaborative !
