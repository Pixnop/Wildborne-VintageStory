# Raccourcis Clavier & Commandes

Ce guide liste tous les raccourcis clavier et commandes importantes ajoutés par les mods de Wildborne.

!!! tip "Configuration"
    La plupart des raccourcis peuvent être modifiés dans **Échap > Paramètres > Contrôles**.

---

## Combat Overhaul

Le mod de combat refonde complètement les contrôles de combat.

| Touche | Action |
|--------|--------|
| `Clic Gauche` | Attaque (maintenir pour charger) |
| `Clic Droit` | Bloquer / Parer |
| `Clic Droit` (maintenu) | Viser (arcs, arbalètes) |
| `R` | Recharger (armes à feu, arbalètes) |
| `Molette` | Changer de mode d'attaque |

!!! warning "Timing"
    Le combat est basé sur le timing. Relâchez l'attaque au bon moment pour des dégâts maximum.

---

## XSkills - Compétences

| Touche | Action |
|--------|--------|
| `K` | Ouvrir le menu des compétences |
| `L` | Ouvrir le menu des classes (Working Classes) |

!!! info "Progression"
    Les compétences augmentent automatiquement en pratiquant l'activité correspondante.

---

## Carry On - Transport

| Touche | Action |
|--------|--------|
| `Shift + Clic Droit` | Soulever un bloc/entité |
| `Clic Droit` | Poser l'objet transporté |
| `Q` | Lâcher l'objet transporté |

!!! tip "Objets transportables"
    Coffres, établis, lits, animaux en cage, et bien plus peuvent être transportés.

---

## Hydrate or Diedrate - Hydratation

| Touche | Action |
|--------|--------|
| `Clic Droit` (sur eau) | Boire directement |
| `Clic Droit` (avec gourde) | Boire de la gourde |
| `Shift + Clic Droit` (sur eau) | Remplir la gourde |

La jauge de soif est affichée à côté de la faim.

---

## Extra Info - Informations

| Touche | Action |
|--------|--------|
| `H` | Basculer l'affichage des infos |
| `Shift + H` | Afficher les infos détaillées |

Affiche des informations détaillées sur les blocs et entités visés.

---

## Prospect Together - Prospection

| Commande | Action |
|----------|--------|
| `.pt` | Afficher les marqueurs de prospection |
| `.pt clear` | Effacer vos marqueurs |
| `.pt toggle` | Activer/désactiver l'affichage |

!!! tip "Partage"
    Les données de prospection sont partagées entre tous les joueurs du serveur.

---

## Auto Map Markers - Carte

| Commande | Action |
|----------|--------|
| `.automapmarkers` ou `.amm` | Paramètres des marqueurs |
| `.amm toggle` | Activer/désactiver les marqueurs auto |

Les points d'intérêt (marchands, ruines, etc.) sont automatiquement marqués sur la carte.

---

## Pet AI - Animaux de compagnie

| Touche | Action |
|--------|--------|
| `Clic Droit` (sur animal apprivoisé) | Menu des commandes |
| `Shift + Clic Droit` | Faire asseoir/lever |

### Commandes vocales (dans le chat)
| Commande | Action |
|----------|--------|
| `sit` | L'animal s'assoit |
| `follow` | L'animal vous suit |
| `stay` | L'animal reste en place |
| `wander` | L'animal se promène librement |

---

## Wolf Taming - Loups

| Action | Comment |
|--------|---------|
| Apprivoiser | Lancer de la viande crue près du loup |
| Commander | `Clic Droit` sur le loup apprivoisé |
| Monter | Certains loups peuvent être montés après apprivoisement complet |

---

## Herbarium - Plantes

| Touche | Action |
|--------|--------|
| `Shift` (en visant une plante) | Afficher les infos de la plante |

---

## Butchering - Dépeçage

| Touche | Action |
|--------|--------|
| `Clic Droit` (avec couteau) | Menu de dépeçage |
| `Sneak + Clic Droit` | Dépeçage rapide |

Utilisez un couteau sur un animal mort pour accéder aux options de dépeçage détaillées.

---

## Animal Cages - Cages

| Action | Comment |
|--------|---------|
| Capturer | `Clic Droit` avec une cage vide sur un petit animal |
| Libérer | `Clic Droit` avec une cage pleine |
| Transporter | Les cages pleines peuvent être transportées avec Carry On |

---

## Primitive Survival - Survie

| Touche | Action |
|--------|--------|
| `Clic Droit` (avec cordage) | Fabriquer des pièges |
| `Sneak + Clic Droit` | Interagir avec les pièges |

---

## Sortable Storage - Tri

| Touche | Action |
|--------|--------|
| `Clic Molette` (sur coffre) | Trier le contenu |
| `Shift + Clic Molette` | Trier par catégorie |

---

## Joy of Sailing - Navigation

| Touche | Action |
|--------|--------|
| `W/A/S/D` | Diriger le bateau |
| `Shift` | Accélérer |
| `Space` | Freiner |
| `E` | Monter/descendre du bateau |

---

## Contrôles Vanilla Utiles

Pour référence, voici les contrôles vanilla importants :

| Touche | Action |
|--------|--------|
| `E` | Inventaire |
| `C` | Inventaire personnage |
| `B` | Menu de fabrication |
| `M` | Carte du monde |
| `Tab` | Liste des joueurs |
| `T` | Chat |
| `/` | Commande |
| `F5` | Vue troisième personne |
| `F6` | Menu debug |

---

## Commandes Serveur Utiles

| Commande | Action |
|----------|--------|
| `/time` | Voir l'heure du jeu |
| `/weather` | Voir la météo |
| `/waypoint add [nom]` | Ajouter un point de passage |
| `/waypoint list` | Liste des points de passage |

---

!!! info "Conflits de touches"
    Si deux mods utilisent la même touche, vous pouvez les reconfigurer dans les paramètres. Certains mods ont aussi leur propre fichier de configuration dans le dossier `ModConfig`.
