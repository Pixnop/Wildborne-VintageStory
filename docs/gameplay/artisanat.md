## Chisel Tools

![Chisel Tools](https://moddbcdn.vintagestory.at/modicon_910cef72d741fec6c488e5c5cf99f84f.png){ align=right width=100 }

La boîte à outils ultime du sculpteur, avec des possibilités créatives quasi infinies.

### Outils spécialisés
| Outil | Fonction |
|-------|----------|
| Pantographe | Copier/coller des designs entre mondes |
| QP Chisel | Mode dessin avec ajout automatique de blocs |
| Rabot | Retirer/ajouter des plans entiers de voxels |
| Coin | Rotation et miroir sur tous les axes |
| Grattoir | Nettoyer les matériaux inutilisés |
| Multigraphe | Combine toutes les fonctions |

### Blocs décoratifs fonctionnels
Transformez vos créations en objets utiles :

- **Échelles** ciselées personnalisées
- **Portes** multi-blocs avec sons configurables
- **Coffres**, lits, foyers, étagères sculptés
- **Panneaux** et enseignes décoratives

## Chiseled Block Retention

Correction essentielle pour les constructeurs créatifs.

- Les blocs ciselés **conservent la chaleur** comme les blocs pleins
- Les pièces avec murs ciselés sont **valides** pour le système de température
- Construisez des intérieurs décorés **sans sacrifier l'isolation**

## Stone Quarry

![Stone Quarry](https://moddbcdn.vintagestory.at/0_logo_da467a7d63ab6cb4f9bd6c56b0fa02d9.png){ align=right width=100 }

Fini le minage fastidieux de pierre. Établissez des carrières pour une production industrielle.

### Chaîne de production
1. Créez une **carrière** pour collecter des dalles de pierre
2. Transformez les dalles en **roches, pierres, briques ou pierre polie**
3. Utilisez le **stockage de gravats** pour broyer en gravier et sable
4. Produisez du **gravier boueux** sur place

### Avantages
- Plus besoin de voyager vers les lacs pour le gravier boueux
- Stockage dédié pour les matériaux de construction
- Compatible avec les outils de différents niveaux

## Millwright

![Millwright](https://moddbcdn.vintagestory.at/logo3_24b54abedee379b6b3825fc526025328.png){ align=right width=100 }

Améliorez vos moulins bien au-delà des capacités vanilla pour une production industrielle.

### Voiles améliorées
| Type | Avantage |
|------|----------|
| Voile centrée | 2x plus efficace que vanilla |
| Voile inclinée | Capture optimale du vent |
| Voile large | Maximum de surface |

### Nouveautés techniques
- **Rotors renforcés** avec plaques de fer (4x plus de voiles)
- **Moulins à axe vertical** pour configurations alternatives
- **Axes de transmission** sans perte à travers murs et plafonds
- **Freins renforcés** pour arrêter les moulins puissants

Combiné avec le mod **Rivers**, placez vos moulins sur les cours d'eau pour une énergie gratuite !

## Smithing Plus

![Smithing Plus](https://moddbcdn.vintagestory.at/smithingpluslogo2_1c117baa69dfb2d780b9fc68ed0b2cf6_480_320.png){ align=right width=100 }

Des améliorations de forge qui rendent le travail du métal plus flexible et réaliste.

### Nouvelles possibilités
- Forgez directement avec des **fragments de métal** chauffés
- Utilisez des **pépites natives** (cuivre, or, argent) sur l'enclume
- **Réparez** les outils cassés : les têtes tombent et peuvent être reforgées
- Récupérez des fragments en **découpant** des objets ou des flèches cassées

### Améliorations d'interface
- **Recherche de recettes** sur l'enclume
- Affichage de la **température** de travail dans les infobulles
- Informations détaillées dans le **manuel** pour chaque métal

## Toolsmith

![Toolsmith](https://moddbcdn.vintagestory.at/ToolsmithBasicLogo_06fd914d5c6806815e1fd9d972e964b6.png){ align=right width=100 }

Les outils deviennent modulaires avec des composants interchangeables et un système d'entretien réaliste.

### Composants d'outils
| Partie | Options |
|--------|---------|
| Tête | Forgée selon le métal |
| Manche | Bâton, os, bois sculpté, bois professionnel |
| Poignée | Ficelle, lin, cuir |
| Traitement | Graisse, cire d'abeille, huile |
| Fixation | Corde, tissu, cuir, clous métalliques |

### Système d'affûtage
- Les outils ont une **jauge d'affûtage** séparée de la durabilité
- Un outil **aiguisé (98%+)** ne s'use pas temporairement
- Un outil **émoussé** s'use deux fois plus vite
- Utilisez **meules et pierres à aiguiser** pour restaurer le tranchant

### Entretien
- **Établi** pour démonter et remonter les outils
- Reforgez les **têtes endommagées** plutôt que de tout refaire
- Chaque composant s'use **indépendamment**
