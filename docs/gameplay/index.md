## Synergies Importantes

!!! info "Combat + Compétences"
    Les mods XSkills s'intègrent avec Combat Overhaul pour des bonus de dégâts basés sur vos compétences.

!!! info "Cuisine + Survie"
    Hydrate or Diedrate ajoute une jauge de soif que vous pouvez gérer avec les boissons de Expanded Foods.
