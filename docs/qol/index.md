## Points Forts

!!! tip "Carry On"
    Le mod **Carry On** vous permet de porter des coffres entiers avec leur contenu ! Idéal pour les déménagements.

!!! tip "Sortable Storage"
    **Sortable Storage** permet de trier automatiquement vos coffres. Un gain de temps considérable !

!!! tip "Player Corpse"
    En cas de mort, **Player Corpse** laisse votre corps avec tout votre inventaire. Retrouvez-le pour récupérer vos affaires !
