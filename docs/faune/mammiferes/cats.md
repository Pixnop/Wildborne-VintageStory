## Description

Le mod **Cats** introduit des chats sauvages et domestiques apprivoisables. Les chatons grandissent en chats adultes, et une fois apprivoisés, vos félins vous aideront à combattre les drifters de bas niveau.

### Fonctionnalités clés
- **Téléportation** derrière vous quand ils ne peuvent pas suivre
- **Étiquettes de nom** visibles en regardant votre chat
- Compatible avec **Primitive Survival** (mangent les filets de poisson cru)

!!! warning "Dépendance requise"
    Ce mod nécessite le mod **Pet AI** pour fonctionner.

## Apprivoisement

1. Trouvez un chat sauvage ou errant
2. Approchez-vous lentement avec du **poisson cru**
3. Donnez-lui à manger régulièrement
4. Le chat deviendra progressivement amical
5. Il vous suivra comme compagnon

!!! tip "Guide en jeu"
    Un guide d'apprivoisement est disponible directement dans le jeu !

---

## Comportement domestiqué

- **Suit** le joueur propriétaire
- **Chasse** les rongeurs et petits animaux
- **Ronronne** quand content
- Peut **dormir** sur les lits
- **Caressable** (clic droit)

---

## Utilité

!!! tip "Anti-nuisibles"
    Les chats chassent automatiquement les rongeurs près de votre base, protégeant vos réserves de nourriture !

!!! info "Combat"
    Les chats apprivoisés peuvent aider contre les drifters de bas niveau.
