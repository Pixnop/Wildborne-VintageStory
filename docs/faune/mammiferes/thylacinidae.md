## Description

Le mod **Fauna of the Stone Age: Thylacinidae Plus** présente **2 espèces** de marsupiaux carnivores nocturnes : le Diable de Tasmanie et le Diable du Queensland. Ces prédateurs pillent les fermes et peuvent grimper pour atteindre leurs proies.

!!! info "Nettoyeurs de carcasses"
    Les diables **recherchent activement les carcasses** du mod FotSA sur de longues distances et les nettoient, permettant le respawn des animaux dans ces zones.

## Comportement

- **Nocturnes** - actifs la nuit
- **Prédateurs** pillant occasionnellement les fermes
- **Mécanique de carcasse** : "Les diables rechercheront les carcasses bloquant les spawns... les nettoyant pour reprendre les spawns normaux"

---

## Domestication

!!! tip "Domestication progressive"
    - Nécessite un mâle et une femelle adultes
    - Les animaux cessent d'attaquer après **10 générations**
    - Ils cessent de fuir après **10 générations**

!!! tip "Observation nocturne"
    Ces créatures sont nocturnes. Cherchez-les la nuit dans les régions tempérées et tropicales.
